Tugas 1 - Dasar Algoritma Pemrograman

1. Contoh Algoritma Dalam Kehidupan Sehari-hari

Mengambil Uang di ATM

  1. Mulai
  2. Masukkan kartu ATM ke mesin ATM.
  3. Pilih bahasa yang diinginkan.
  4. Masukkan PIN ATM.
  5. Jika PIN benar, lanjut ke menu utama.
  6. Pilih menu Tarik Tunai.
  7. Pilih jumlah uang yang akan diambil.
  8. Mesin memproses transaksi.
  9. Ambil uang yang keluar dari mesin.
  10. Ambil kartu ATM.
  11. Transaksi selesai.

Membuat Kopi Dengan Rasa Manis dan Pahit Yang Pas

  1. Siapkan gelas, kopi, gula, dan air panas.
  2. Masukkan 1 sendok kopi ke dalam gelas.
  3. Tambahkan 1–2 sendok gula.
  4. Tuangkan air panas ke dalam gelas.
  5. Aduk hingga larut.
  6. Cicipi kopi.
  7. Jika terlalu pahit tambahkan gula.
  8. Jika terlalu manis tambahkan sedikit kopi.
  9. Aduk kembali hingga rasa pas.
  10. Kopi siap diminum.

Menghitung Nilai y = 3x + 6

  1. Masukkan nilai x.
  2. Hitung y = 3 × x + 6.
  3. Tampilkan nilai y.
  4. Selesai.

2. Pengertian Algoritma dan Program

Algoritma adalah urutan langkah-langkah logis dan sistematis untuk menyelesaikan suatu masalah.

Program adalah kumpulan instruksi yang ditulis dalam bahasa pemrograman yang digunakan komputer untuk menjalankan suatu algoritma.

  • Algoritma = langkah penyelesaian masalah.
  • Program = implementasi algoritma dalam bahasa pemrograman.

3. Tiga Struktur Dasar Algoritma

Sekuensial (Sequence)

Struktur yang menjalankan langkah-langkah secara berurutan dari awal sampai akhir.

  • Masukkan angka
  • Hitung hasil
  • Tampilkan hasil

Pemilihan (Selection)

Struktur yang digunakan untuk memilih aksi berdasarkan kondisi tertentu.

  • Jika nilai ≥ 75 → Lulus
  • Jika nilai < 75 → Tidak Lulus

Perulangan (Looping)

Struktur yang digunakan untuk mengulang suatu proses sampai kondisi terpenuhi.

  • Tampilkan angka
  • Tambah angka
  • Ulangi sampai angka = 5

4. Fungsi Bahasa Pemrograman

  • Sebagai alat komunikasi dengan komputer.
  • Membuat aplikasi atau program.
  • Mengolah data.
  • Mengotomatisasi pekerjaan.
  • Mengontrol perangkat keras (hardware).

Tugas 2 - Tipe Data dan Operator Java

1. Perbedaan Tipe Data Primitif dan Non-Primitif

Tipe Data Primitif

  • Tipe data bawaan (built-in) dalam Java.
  • Menyimpan nilai langsung.
  • Ukurannya tetap.
  • Tidak memiliki method (fungsi).
  • Contoh: int, char, boolean, double, float, byte, short, long.

Tipe Data Non-Primitif

  • Tipe data yang dibuat dari class.
  • Menyimpan referensi atau alamat memori.
  • Bisa memiliki method.
  • Ukurannya tidak tetap (dinamis).
  • Contoh: String, Array, Object, dan Class.

2. Analisis Kesalahan Penulisan Variabel

Int 5angka;
String nama siswa;
Int static;

Kesalahan

  • Int 5angka;
    Salah karena tipe data harus ditulis int (huruf kecil) dan nama variabel tidak boleh diawali angka.
  • String nama siswa;
    Salah karena nama variabel tidak boleh mengandung spasi.
  • Int static;
    Salah karena int harus huruf kecil dan static merupakan keyword Java sehingga tidak boleh digunakan sebagai nama variabel.

3. Contoh Deklarasi Array

int[] angka = new int[5];

Atau:

int angka[] = {1, 2, 3, 4, 5};

Penjelasan

  • int[] → tipe data array.
  • angka → nama variabel.
  • [5] → jumlah elemen array.

4. Operator Aritmatika Dalam Java

Operator Keterangan
+ Penjumlahan
- Pengurangan
* Perkalian
/ Pembagian
% Modulus (sisa hasil bagi)

Contoh:

  • a + b → menjumlahkan nilai.
  • a % b → menghasilkan sisa pembagian.

5. Implementasi Operator Pembagian Dalam Java

public class Pembagian {

    public static void main(String[] args) {

        int a = 10;
        int b = 2;

        int hasil = a / b;

        System.out.println(
            "Hasil pembagian: " + hasil
        );

    }

}

Output

Hasil pembagian: 5

Catatan

  • Jika menggunakan int, hasil pembagian akan dibulatkan.
  • Untuk hasil desimal gunakan double.
double hasil = 10.0 / 3;

Tugas 3 - Struktur Algoritma Sekuensial

1. Pengertian Instruksi Pada Algoritma

Instruksi pada algoritma adalah perintah-perintah atau langkah-langkah yang harus dilakukan secara sistematis untuk menyelesaikan suatu masalah.

Instruksi ditulis secara jelas dan berurutan agar dapat dipahami serta dijalankan oleh manusia maupun komputer.

2. Pengertian Struktur Algoritma Sekuensial

Struktur algoritma sekuensial adalah bentuk algoritma yang langkah-langkahnya dijalankan secara berurutan dari awal hingga akhir tanpa adanya percabangan maupun perulangan.

Setiap instruksi dijalankan satu per satu sesuai urutan penulisannya.

3. Karakteristik Struktur Sekuensial

  • Langkah-langkah dieksekusi secara berurutan.
  • Tidak ada percabangan (if/else).
  • Tidak ada perulangan (loop).
  • Alur program sederhana dan mudah dipahami.
  • Setiap langkah hanya dijalankan satu kali.

4. Contoh Penggunaan Struktur Sekuensial

Menghitung Luas Persegi Panjang

  1. Mulai.
  2. Masukkan nilai panjang.
  3. Masukkan nilai lebar.
  4. Hitung luas = panjang × lebar.
  5. Tampilkan hasil luas.
  6. Selesai.
Input:
Panjang = 10
Lebar = 5

Proses:
Luas = Panjang × Lebar
Luas = 10 × 5

Output:
Luas = 50

Contoh ini menunjukkan bahwa setiap langkah dilakukan secara berurutan tanpa adanya percabangan maupun perulangan.

Tugas 4 - Algoritma Pemrograman

Tugas 4 - Algoritma Pemrograman: Flowchart & Implementasi

1. Notasi Kalimat Deskriptif

Deklarasi:

jenis: String, lama: Integer, tarif: Integer, total: Integer.

Deskripsi:

  • Mulai program.
  • Masukkan jenis kendaraan.
  • Masukkan lama parkir dalam jam.
  • Cek Kondisi: Jika jenis = "Mobil", maka tarif = 5000, jika tidak tarif = 2000.
  • Hitung total = lama * tarif.
  • Tampilkan total bayar.
  • Selesai.

2. Notasi Pseudocode

Program HitungParkir

Deklarasi:
    jenis : string
    lama, tarif, total : integer

Algoritma:
    read (jenis)
    read (lama)

    IF jenis == "Mobil" THEN
        tarif = 5000
    ELSE
        tarif = 2000
    ENDIF

    total = lama * tarif
    write (total)

END

4. Implementasi Menggunakan Python

# PROGRAM MENGHITUNG BIAYA PARKIR
jenis = input("Masukkan jenis kendaraan (Motor/Mobil): ")
lama = int(input("Masukkan lama parkir (jam): "))

if jenis.lower() == "mobil":
    tarif = 5000
else:
    tarif = 2000

total = lama * tarif

print("Total biaya parkir adalah: Rp", total)

# OUTPUT HASIL RUNNING:
    - Mobil : 2 jam*5.000 = 10.000
    - Motor : 3 jam*2.000 = 6.000

Tugas 5 - Struktur Perulangan Dalam Python

1. Perulangan Struktur For

Perulangan For digunakan untuk mengulang suatu perintah berdasarkan jumlah perulangan yang telah ditentukan.

Kode Python


for i in range(1, 6):
    print("Perulangan ke-", i)

Hasil Running

Perulangan ke- 1
Perulangan ke- 2
Perulangan ke- 3
Perulangan ke- 4
Perulangan ke- 5

2. Perulangan Struktur While

Perulangan While akan berjalan selama kondisi yang diberikan bernilai benar (True).

Kode Python


i = 1

while i <= 5:
    print("Perulangan ke-", i)
    i += 1

Hasil Running

Perulangan ke- 1
Perulangan ke- 2
Perulangan ke- 3
Perulangan ke- 4
Perulangan ke- 5

3. Perulangan Struktur Do-While

Bahasa Python tidak memiliki struktur Do-While secara langsung, sehingga dapat dibuat menggunakan while True.

Kode Python


i = 1

while True:
    print("Perulangan ke-", i)
    i += 1

    if i > 5:
        break

Hasil Running

Perulangan ke- 1
Perulangan ke- 2
Perulangan ke- 3
Perulangan ke- 4
Perulangan ke- 5

4. Perulangan Bersarang (Nested Loop)

Nested Loop adalah perulangan yang berada di dalam perulangan lain.

Kode Python


for i in range(5):
    for j in range(5):
        print("*", end=" ")
    print()

Hasil Running

* * * * *
* * * * *
* * * * *
* * * * *
* * * * *

Kesimpulan

Struktur perulangan digunakan untuk menjalankan suatu instruksi secara berulang tanpa harus menuliskan kode yang sama berkali-kali. Dalam Python terdapat berbagai jenis perulangan seperti For, While, Do-While (simulasi), dan Nested Loop yang masing-masing memiliki fungsi sesuai kebutuhan program.

Tugas 6 - Pretest Array

1. Apa yang Dimaksud Dengan Array?

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data yang memiliki tipe data yang sama dalam satu variabel. Setiap data dalam array disebut elemen dan memiliki indeks untuk mengaksesnya.

Contoh:


nilai = [80, 85, 90, 75, 88]

2. Jelaskan Yang Dimaksud Dengan Array 1 Dimensi!

Array 1 dimensi adalah array yang hanya memiliki satu indeks untuk mengakses datanya. Data disimpan secara berurutan seperti daftar atau list.

Contoh:


buah = ['Apel', 'Mangga', 'Jeruk', 'Pisang']
buah[0]

Hasil: 'Apel' karena indeks dimulai dari 0.

3. Bagaimana Cara Mendeklarasikan Array?

Mendeklarasikan array berarti membuat array dengan menentukan nama array dan data yang akan disimpan.

Contoh Python:


angka = [10, 20, 30, 40, 50]

Contoh C++:


int angka[5] = {10,20,30,40,50};

4. Tipe Data Yang Digunakan Untuk Mendeklarasikan Array

  • Integer (int) 
    umur = [18,19,20]
  • Float 
    tinggi = [160.5, 165.2, 170.8]
  • String 
    nama = ['Andi','Budi','Siti']
  • Character (C++) 
    char huruf[4] = {'A','B','C','D'};
  • Boolean 
    status = [True, False, True]

5. Macam-Macam Array Yang Diketahui

a. Array Satu Dimensi (1D)


nilai = [80, 85, 90, 75]

b. Array Dua Dimensi (2D)


matriks = [[1,2,3],[4,5,6]]

c. Array Multidimensi


data = [[[1,2],[3,4]]]

Kesimpulan

Array adalah struktur data yang digunakan untuk menyimpan banyak data bertipe sama dalam satu variabel. Berdasarkan dimensinya, array dibedakan menjadi array satu dimensi, dua dimensi, dan multidimensi. Penggunaan array membuat pengelolaan data menjadi lebih mudah dan efisien.

Tugas 7 - Array Dua Dimensi

1. Apa yang Kamu Ketahui Tentang Array Dua Dimensi?

Array dua dimensi adalah struktur data yang digunakan untuk menyimpan sekumpulan data yang tersusun dalam bentuk baris (row) dan kolom (column) sehingga menyerupai tabel atau matriks.

Contoh Python


nilai = [
    [80, 85, 90],
    [75, 88, 92],
    [70, 78, 85]
]

2. Bagaimana Cara Mendeklarasikan Array Dua Dimensi?

Mendeklarasikan array dua dimensi berarti membuat array yang terdiri dari beberapa baris dan kolom.

Contoh Python


matriks = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

3. Selain Array Dua Dimensi, Sebutkan Macam-Macam Array!

a. Array Satu Dimensi (1D)


angka = [10, 20, 30, 40, 50]

b. Array Dua Dimensi (2D)


matriks = [[1,2],[3,4]]

c. Array Tiga Dimensi (3D)


data = [[[1,2],[3,4]]]

d. Array Multidimensi


data = [[[1,2],[3,4]], [[5,6],[7,8]]]

4. Apa Perbedaan Array Satu Dimensi Dengan Array Dua Dimensi?

  • Array satu dimensi hanya memiliki satu indeks.
  • Bentuknya seperti daftar (list).
  • Data disusun dalam satu baris.
  • Lebih sederhana untuk digunakan.

  • Array dua dimensi memiliki dua indeks.
  • Bentuknya seperti tabel.
  • Data disusun dalam baris dan kolom.
  • Cocok untuk menyimpan data yang lebih kompleks.

5. Apa Perbedaan Array Dua Dimensi Dengan Array Multidimensi?

  • Array dua dimensi memiliki dua indeks (baris dan kolom).
  • Bentuknya menyerupai tabel atau matriks.
  • Lebih mudah dipahami dan digunakan.

  • Array multidimensi memiliki lebih dari dua indeks.
  • Dapat menyimpan data yang lebih kompleks.
  • Biasanya digunakan pada aplikasi yang membutuhkan banyak dimensi data.

Kesimpulan

Array adalah struktur data yang digunakan untuk menyimpan banyak data dalam satu variabel. Array dua dimensi menyimpan data dalam bentuk baris dan kolom seperti tabel. Selain array dua dimensi, terdapat array satu dimensi, tiga dimensi, dan multidimensi. Penggunaan array memudahkan pengelolaan data sehingga program menjadi lebih terstruktur dan efisien.
